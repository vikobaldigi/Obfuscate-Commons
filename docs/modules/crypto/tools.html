Core Utilities

  • GPG: Full-featured key management, encryption, and signing suite for files and email.
  • age: A simpler modern alternative to GPG, focused on secure file encryption.
  • OpenSSL: A powerful library for command-line crypto and certificate operations.
  • Keybase: Combines identity verification, file sharing, and chat ‚Äî all encrypted end-to-end.

Encrypted Messaging

  • Signal: Free, open source, end-to-end encrypted calling and messaging app.
  • Session: Decentralized Signal fork with no phone numbers or central servers.
  • Element: Matrix-based messaging with decentralization and federation.

These tools make cryptography invisible to the user ‚Äî but they all rely on strong key exchange and forward secrecy.

File & Disk Encryption

  • Veracrypt: Create encrypted volumes or encrypt whole disks.
  • LUKS: Standard for Linux disk encryption.
  • BitLocker / FileVault: Built-in disk encryption for Windows and macOS.

Encryption at rest protects your data if a device is lost or stolen. But it‚Äôs only as strong as the password used to unlock it.

Browser and Metadata Privacy

  • Tor Browser: Routes your traffic through a global anonymity network.
  • Tails OS: Live operating system that routes all traffic through Tor.
  • uBlock Origin, Privacy Badger: Prevent trackers and third-party scripts.

Encryption is more than scrambling text. It's about shaping the entire surface of your digital exposure.
