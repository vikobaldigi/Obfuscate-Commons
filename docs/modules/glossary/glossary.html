Glossary of Terms

Key terms and concepts introduced throughout Obfuscate-OS.

Asymmetric Encryption
A form of encryption that uses a public/private key pair. The encryption key can be widely shared (public), while the decryption key must be kept private. This model enables secure communication without needing a shared secret in advance — solving the key distribution problem.
Block Cipher
A method of encrypting data in fixed-size blocks (e.g., AES) rather than as a continuous stream.
Blockchain
A distributed ledger of cryptographically linked data blocks, maintained across multiple systems without a central authority.
Command-Line Interface (CLI)
A text-based interface used to interact with a computer system by entering commands.
Encryption
The process of converting readable data into ciphertext to prevent unauthorized access.
Hash Function
A mathematical function that converts data into a fixed-length hash. Used for verification and integrity checking.
Keypair
In asymmetric cryptography, a pair of mathematically related keys — one public, one private.
Metadata
Metadata is data about data — like who sent a message, when, from where, and to whom. It structures visibility and is often more revealing than content itself. Surveillance systems rely on metadata to model relationships and behaviors.
Obfuscation
Obfuscation is the deliberate addition of misleading, noisy, or ambiguous information to confuse surveillance or analysis. It resists clarity by flooding systems with plausible alternatives — buying time, hiding meaning, or resisting profiling.
POSIX
A family of standards for maintaining compatibility between operating systems. Includes a common set of command-line tools.
Public-Key Cryptography
Public-key (or asymmetric) cryptography enables anyone to encrypt a message using a recipient's public key, but only the recipient can decrypt it with their private key. This unlocks secure, decentralized communication, digital signatures, and modern protocols like HTTPS.
Smart Contract
Self-executing code stored on a blockchain that runs when predefined conditions are met.
Stream Cipher
Encrypts data one bit or byte at a time, often used in real-time applications.
Symmetric Encryption
A cryptographic system where the same key is used for both encryption and decryption. It's fast and efficient, but it requires secure key sharing ahead of time. Examples include AES and ChaCha20.
Terminal
A text-based environment for controlling an operating system, offering direct system access beyond graphical interfaces.
