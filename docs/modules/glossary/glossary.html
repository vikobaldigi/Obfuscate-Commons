AES (Advanced Encryption Standard) CPU
Symmetric cipher widely used in digital security, known for speed and resilience.
Asymmetric Encryption CPU
Uses a public/private keypair to allow secure communication without a pre-shared secret.
Block Cipher CYPHER
Encrypts data in fixed-size blocks rather than one bit or byte at a time.
Blockchain BLOCKCHAIN
A tamper-resistant digital ledger built from cryptographically linked blocks.
Certificate Authority (CA) KEYPAIR
An entity that issues digital certificates to confirm identities in public-key infrastructures.
ChaCha20 CPU
A stream cipher known for its speed and use in mobile-friendly encryption systems.
Cipher CPU
An algorithm or technique that transforms plaintext into ciphertext ‚Äî the core mechanism behind encryption.
CLI (Command-Line Interface) GUIDE
Text-based environment for issuing commands directly to a system.
Consensus Mechanism BLOCKCHAIN
System for agreeing on blockchain data (e.g., Proof of Work or Proof of Stake).
Cryptanalysis CPU
The study of breaking or weakening encryption systems.
Cryptographic Failure SEC
When encryption fails due to poor implementation, not flaws in the math itself.
Cryptographic Hash BLOCKCHAIN
Function that maps input data to a fixed output ‚Äî used for integrity checks and chaining blocks.
DAO (Decentralized Autonomous Organization) BLOCKCHAIN
A self-governing group coordinated via smart contracts on a blockchain.
DES (Data Encryption Standard) SEC
An obsolete encryption algorithm vulnerable to brute force attacks.
Digital Identity BLOCKCHAIN
Online identity credentials secured with cryptographic keys or blockchain systems.
Digital Signature KEYPAIR
A cryptographic marker verifying the authenticity of a message or transaction.
ECC (Elliptic Curve Cryptography) CPU
A compact, efficient form of public-key encryption based on algebraic curves.
Encryption CPU
Conversion of readable data (plaintext) into unreadable form (ciphertext).
Enigma CYPHER
WWII-era rotor-based cipher machine used by Germany; famously broken by Allied cryptanalysis at Bletchley Park.
Frequency Analysis CYPHER
A method to break ciphers by analyzing character occurrence patterns.
Hash Function BLOCKCHAIN
Generates a short string (hash) from data; used in integrity checks and blockchains.
Keypair KEYPAIR
One public key and one private key used in asymmetric encryption.
Man-in-the-Middle Attack KEYPAIR
When an attacker intercepts communication between two parties without detection.
MD5 SEC
Deprecated hash function with known vulnerabilities; avoid in new projects.
Merkle Tree BLOCKCHAIN
A hash-based structure that efficiently verifies data consistency in blockchains.
Metadata NET
Descriptive information about data ‚Äî like time, sender, and location.
Obfuscation CPU
The deliberate introduction of ambiguity to resist analysis or surveillance.
Proof of Stake BLOCKCHAIN
Consensus model where users stake tokens to validate blockchain transactions.
Proof of Work BLOCKCHAIN
Consensus model requiring computation to validate blockchain transactions.
Public-Key Cryptography KEYPAIR
Encryption system using public/private keypairs to share secrets securely.
RSA CPU
Early public-key encryption system based on prime factorization.
Smart Contract BLOCKCHAIN
Code that executes on a blockchain when conditions are met ‚Äî used in DAOs and apps.
Symmetric Encryption CPU
Uses a single key to both encrypt and decrypt data. Requires secure key sharing.
Terminal GUIDE
A command-line interface to control and inspect systems without a GUI.
