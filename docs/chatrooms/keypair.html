KEYPAIR

✷
Twin stars of public and private keys, bound in mathematical harmony. The foundation of asymmetric encryption, where one key locks and the other unlocks.

The Duality

In the realm of cryptography, key pairs are like two sides of the same coin. The public key can be shared with anyone, while the private key must be kept secret. This duality enables secure communication without prior key exchange.

How It Works

When you want to send a secure message:

  1. Use the recipient's public key to encrypt the message
  2. Only their private key can decrypt it
  3. Even if someone intercepts the message, they cannot read it without the private key

Example

// Generate a key pair
const { publicKey, privateKey } = await window.crypto.subtle.generateKey(
  {
    name: "RSA-OAEP",
    modulusLength: 2048,
    publicExponent: new Uint8Array([1, 0, 1]),
    hash: "SHA-256",
  },
  true,
  ["encrypt", "decrypt"]
);
